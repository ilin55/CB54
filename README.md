# CB54
Files with various data on CB54 field

CB54_table2_N.xlsx -- observational data for about 900 stars in the field A at CB54 globule
The table includes: our names of the stars, Gaia DR3 numbers and coordinates, magnitudes in B, V, R bands with errors.

CB54_table3_N.xlsx -- observational data for about 100 stars in the field b at CB54 globule
The table includes: our names of the stars, Gaia DR3 numbers and coordinates, magnitudes in B, V, R bands with errors.
